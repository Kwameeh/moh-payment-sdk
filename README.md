# MOH Payment SDK

Monorepo for the MOH Payment Gateway SDK.

## Packages

| Package | Description |
|---------|-------------|
| `@moh/payment-sdk` | Main SDK (vanilla JS + React) |

## Development

```bash
# Install dependencies
npm install

# Build
cd packages/payment-sdk && npm run build
```

## Publishing

```bash
cd packages/payment-sdk
npm publish
```
